Week 3 - Notes

This

methods can be added to objects by declaring it first then adding to object or by declaring inside object.

this refers to the object before the dot.

You can use the same function with different objects

Objects

javascript objects are more similar to a hash than OOP languages' object

created with curly braces

use in to check if an object has a property use for in to find all objects properties

keys() will return array of keys, values() will do the same for values, entries() for array of both

use delete to remove properties

Objects can contain other objects

Objects are passed by reference

Objects can be passed as a parameter to a function

use objects to prevent repeating names in same scope

JSON.parse() converts Json to JS object stringify() does the reverse, methods are ignored

the Math object has many useful methods and constants

there is a build in Date object, use get methods to return info

Regular Expression

RegExp used to search for strings that match ua pattern

use // to write a RegExp

test() returns bool based on if the pattern is in the string

exec() returns first match found

[] represent any one character sequences can use -

^ negates the Expression

other metacharacters and modifiers can be used to refine your search

DOM - Document Object Model

DOM - represents an HTML as a network of connected nodes

document - a built in object

Commonly used elements

  • .body
  • .images
  • .links
  • .anchors
  • .forms

getElementsByTagName() returns list of nodes with that tag

querySelector() allows you to use CSS notation to return first element that matches

Finding Value of a node

  • .firstChild.nodeValue
  • .textContent

retrieve elements attributes using getAttribute() or dot notation

set them with setAttribute (use ClassList to avoid overwriting other classes)

createTextNode() to create TextNode

appendChild() to add node as chide to node

insertBefore() adds element before element

innerHTML returns all raw HTML can be used to add to the document

use style property to update CSS

Event Listener

event listener allows the program continue processing the rest of the code without continually checking for the event

don't add JavaScript inside the Html its messy like using onclick in tags

instead use .addEventListener() the first parameter is the event i.e click the second parameter is the function you want to invoke

target property returns reference to the element that fired the event

don't use click and double click on the same element it will fire twice

ScreenX and Y tells pixel location of event on screen

clientX and Y tell number of pixels being used

pageX and Y tells pixel location of event in Document

Possible Events

  • keydown
  • keypress
  • keyup
  • touchstart - use cautiously
  • touchend
  • touchmove
  • touchenter
  • touchcancel

bubbling - bubbles up to parent happens by default

Capturing - set with third parameter of addEventListener propagates to child
