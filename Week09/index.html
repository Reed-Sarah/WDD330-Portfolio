Week 9 - Notes

The Window Object

functions and methods created in the global scope belong to the window object

BOM - (Browser Object Model) contains info about browser and screen

global variables are created without const, let or var

alert, confirm, prompt are all methods of the window object

window.location.href returns the URL and can be used to change it

window.history used to access info about previously visited pages

window.open opens a new window with provided URL

window.screen provides information about screen size

document.write() will rewrite page to string passed in but should rarely if ever be used

cookies can only be read by the domain that set them

use document.cookie to append name values to the end of the cookie

cookies are deleted at the end of the session by default, but you can add expires=date to make them last longer

window.setTimeout() accepts a function and number of milliseconds used for scheduling execution

window.setInterval() is similar but repeats call to function after each time interval passed in

HTML 5 APIs

data- allows you to embed data into a webpage with custom attributes

access the attribute with dataset.whateveryouputafterthehypen

use localStorage.setItem() to set name value pair

localStorage.getItem to retrieve value passing in a name you can also use dot notation

localStorage.removeItem to remove also can use delete operator

localStorage.clear() removes everything stored into local localStorage

geolocation API obtains the geographical location of the device user must grant access

is a property of the navigator object

getCurrentPosition will return a position object which can be used to find, speed, altitude, heading (direction device is heading), and a timestamp

watchPosition will call a callback every time position is updated

web workers allow for concurrency JS

use Worker constructor to use

Websocket allows two way communication with the server i.e chats

Notification API allows you to show messages using the system's notifications need permission from user

Audio and video tags can be used with src attributes to play sounds

can be referenced in js using getElementbyTagName .play will start or resume video .pause will pause video

Shims and Polyfills are libraries that fill in the missing code when using APIs that are not supported
